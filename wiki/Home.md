# Jamlog Wiki

Docs for [fast.jamlog.lol](https://fast.jamlog.lol), the Last.fm now-playing overlay you lay out in a drag-and-drop editor and paste into OBS as a browser source.

- **[Custom CSS](Custom-CSS)** (experimental): write real CSS against your widget for gradients, animations, borders, and anything else the editor doesn't have a button for.
